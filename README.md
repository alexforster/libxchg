## libxchg

Generic IPC library with rich messaging semantics.

**Version:** 1.0.0<br/>
**Author:** Alex Forster \<alex@alexforster.com\><br/>
**License:** [Apache-2.0](https://www.apache.org/licenses/LICENSE-2.0)<br/>

### Feature Highlights

 * Lockfree
 * Zero-copy
 * Zero allocations
 * Comprehensive testing
 * Comprehensive fuzzing
